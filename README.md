### 常见十大经典算法

#### 时间复杂度

有关时间复杂度的定义，可以在百度、google上查阅，这里只为留一个印象。

`Ο(1)` 表示基本语句的执行次数是一个常数，一般来说，只要算法中不存在循环语句，其时间复杂度就是 `Ο(1)` 。其中`Ο(log<sub>2</sub>n)`、`Ο(n)`、 `Ο(nlog2n)`、`Ο(n2)` 和 `Ο(n3)` 称为多项式时间，而 `Ο(2n)` 和 `Ο(n!)` 称为指数时间。

常见的算法时间复杂度由小到大依次为：`Ο(1)`＜`Ο(log2n)`＜`Ο(n)`＜`Ο(nlog2n)`＜`Ο(n2)`＜`Ο(n3)`＜…＜`Ο(2n)`＜`Ο(n!)`
